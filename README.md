# ☔️ 아 맞다 우산! (Ah-Mat-Da! Umbrella)

> **IP 위치 기반 날씨 데이터와 임베디드 센서 제어를 결합한 자동 우산 디스펜서 시스템**
> 본 프로젝트는 부경대학교 정보통신공학전공 임베디드시스템설계 과목의 6조 미니 캡스톤 프로젝트 결과물입니다.

---

## 📌 1. 프로젝트 개요 & 기획 배경
바쁜 일상 속에서 외출할 때 비 소식을 미처 확인하지 못해 우산을 챙기지 않거나, 반대로 비가 오지 않는데도 불필요하게 우산을 들고 나가는 불편함이 자주 발생합니다.

**'아 맞다 우산!'** 시스템은 이러한 일상적인 문제를 해결하기 위해 기획되었습니다. 평소에는 대기 상태를 유지하다가 사용자가 외출을 위해 현관문에 다가오면, 시스템이 실시간 날씨와 예보 데이터를 조회합니다. 당일 단 몇 시간이라도 비 예보가 있다면 자동으로 우산 통 문을 열고 우산을 기울여주어 사용자가 직관적으로 우산을 챙길 수 있도록 돕는 시스템입니다.

---

## 🛠 2. 시스템 구성 및 개발 환경
### Development Environment
- **OS:** Linux (Raspberry Pi OS)
- **Language:** Python 3

### Hardware Component
- **Main Board:** Raspberry Pi 3
- **초음파 센서:** 현관문에 접근하는 사용자 실시간 감지 (0.5초 주기)
- **OLED 디스플레이 모듈:** 기상 상태 및 안내 메시지 시각화 (날씨 정보 표시부)
- **서보 모터 (2개):**
  - **모터 1:** 우산 통 문 개방 (90도 제어)
  - **모터 2:** 우산 통 기울임 (60도 제어)

---

## ✨ 3. 주요 핵심 기능

### 1) IP 기반 실시간 위치 탐색 및 On-Demand API 호출
- 기기가 구동되면 접속된 네트워크의 IP를 이용해 현재 위치의 위도(lat)와 경도(lon)를 자동으로 구합니다.
- 불필요한 API 호출을 방지하기 위해, 초음파 센서에 **사용자가 감지되었을 때만** OpenWeatherMap API에 해당 좌표의 날씨 데이터를 요청합니다.

### 2) 0.5초 주기의 초음파 센서 감시 로직
- 시스템은 **0.5초마다** 초음파 센서를 통해 사용자가 접근하는지 지속적으로 확인합니다.

### 3) 예상 강수 확률 기반 판독
- API에서 받아온 데이터를 바탕으로 오늘 예상 강수 확률을 확인합니다.
- 하루 중 **단 몇 시간이라도 강수 확률이 존재한다면 '비 예보가 있다'고 판단**하여 확실하게 우산을 챙길 수 있도록 보수적인 알림 알고리즘을 적용했습니다.

### 4) 서보 모터 순차 제어 메커니즘
- 비 예보가 있을 시 두 개의 서보 모터가 순차적으로 작동합니다. 첫 번째 모터가 **우산 통 문을 90도로 개방**하고, 연이어 두 번째 모터가 **우산 통을 60도로 기울여** 사용자가 손쉽게 우산을 집을 수 있게 한 후 다시 제자리로 복귀합니다.

## ⚙️ 4. 시스템 동작 프로세스

```text
[1. 시스템 구동]
       ↓
[2. 현재 위치 탐색: IP를 이용하여 현재 위치의 위도(lat), 경도(lon) 계산]
       ↓
[3. 접근 감지: 0.5초 주기로 초음파 센서를 통해 사용자 접근 여부 확인]
       ↓
┌───────────────── 사용자가 접근했을 때 (초음파 감지) ───────────────────┐
 [4. 날씨 요청: OpenWeatherMap API에 날씨 데이터 요청]                
        ↓                                                             
 [5. 정보 출력: 날씨 상태에 따라 OLED 화면에 그래픽/텍스트 표시]
   
        ↓                                                             
 [6. 강수 판단: 오늘 예상 강수 확률 계산                              
                (단 몇 시간이라도 확률이 있으면 비 예보로 판단)]      
                                                                      
        비 예보가 있는 경우                           비 예보 없는 경우
                ↓                                           ↓
   [7. 모터 1] 우산 통 문 개방 (90도)                 일반 정보 표시 유지
   [8. 모터 2] 우산 통 기울임 (60도)                  (모터 미동작)
   [9. 복귀] 서보 모터 다시 제자리로 원위치                           
                                                                      
└───────────────────────────────────────────────────────────────────────┘
